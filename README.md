## 🚀 Getting Started

### Prerequisites
- Node.js 20+ 
- pnpm / npm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kumailkmr/kumailkmr-portfolio.git
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Build for production:
   ```bash
   pnpm build
   ```

## 🛠️ Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components/animations/premium`: Core 3D background components.
- `src/components/shared`: Reusable premium UI components.
- `src/lib/data.ts`: Central source of truth for portfolio content.
- `src/lib/utils.ts`: Tailwind CSS utility helpers.
